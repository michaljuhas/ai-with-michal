| Field | Use for |
|---|---|
| `firstName` | Personalisation |
| `companyName` | Personalisation |
| `jobTitle` | Segment verification (filter leads before import) |
| Country (custom field or filter) | Segment gating |
| Industry (custom field) | Segment gating |
| Company size (custom field) | Segment gating |

Import leads as CSV with columns: `firstName`, `lastName`, `email`, `companyName`, `jobTitle`
Add custom fields: `country`, `industry`, `companySize` — useful for filtering stats later.
