**3. Build + handoff** — I build the integration, document it, and train your team so they can maintain and extend it themselves.

I work with a small number of clients at a time — so the work is hands-on, not delegated to a junior.

## LemList UI Setup Instructions

For each campaign:

1. Open the campaign in LemList → **Sequence** tab
2. Click **+ Add step** → choose **Email**
3. Set **delay** to 0 for Email 1, 2 for Email 2, 5 for Email 3 (business days)
4. Paste subject and body from above
5. Enable **{{firstName}}** and **{{companyName}}** personalization (LemList will use lead fields)
6. Set the **sender** to michal@michaljuhas.eu (same as existing campaigns)
7. Once steps are added, set campaign **status → Active** — new inbound leads will auto-enroll via the API hook
